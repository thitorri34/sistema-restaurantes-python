


# Sistema de Restaurantes em Python

Projeto desenvolvido para praticar **Programação Orientada a Objetos (POO)** em Python.
O sistema simula o cadastro de restaurantes e permite registrar avaliações de clientes, além de calcular automaticamente a média das avaliações.

---

## 📌 Funcionalidades

* Cadastro de restaurantes
* Ativação e desativação de restaurantes
* Registro de avaliações de clientes
* Cálculo automático da média das avaliações
* Listagem de restaurantes com suas informações

---

## 🧱 Estrutura do Projeto

```
sistema-restaurantes-python
│
├── app.py
│
└── Modelos
    ├── __init__.py
    ├── Restaurante.py
    └── Avaliacao.py
```

### Classes principais

**Restaurante**

* Representa um restaurante do sistema
* Armazena nome, categoria, status e avaliações
* Calcula a média das avaliações
* Permite alternar o estado ativo/inativo

**Avaliacao**

* Representa a avaliação feita por um cliente
* Guarda o nome do cliente e a nota atribuída

## ▶️ Como executar o projeto

1. Clone o repositório

```
git clone https://github.com/thitorri34/sistema-restaurantes-python.git
```

2. Entre na pasta do projeto

```
cd sistema-restaurantes-python
```

3. Execute o programa

```
python app.py
```
